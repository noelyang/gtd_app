## 📱 什么是 PWA？

PWA (Progressive Web App) 是一种现代 Web 应用技术，让网站可以像原生应用一样使用。

### GTD Focus 的 PWA 特性
- ✅ **离线使用** - 无网络时也能查看和编辑任务
- ✅ **添加到主屏幕** - 像原生应用一样启动
- ✅ **全屏显示** - 无浏览器地址栏
- ✅ **快速加载** - 缓存加速
- ✅ **自动更新** - 后台静默更新

## 🚀 安装到设备

### iOS (Safari)

#### 步骤:
1. 打开 GTD Focus 网站
2. 点击 Safari 底部 **分享** 按钮 (方框 + 箭头)
3. 向下滑动，点击 **"添加到主屏幕"**
4. 点击右上角 **"添加"**
5. 主屏幕出现 GTD 图标

#### 特点:
- 图标出现在主屏幕
- 点击图标全屏打开
- 无 Safari 浏览器界面
- 支持离线使用

### Android (Chrome)

#### 步骤:
1. 打开 GTD Focus 网站
2. 点击 Chrome 菜单 (三个点)
3. 点击 **"添加到主屏幕"**
4. 确认应用名称
5. 点击 **"添加"**
6. 主屏幕出现 GTD 图标

**或者:**
- 浏览器地址栏出现安装图标
- 直接点击安装

#### 特点:
- 图标出现在主屏幕
- 点击图标全屏打开
- 支持离线使用
- 可能出现在应用抽屉

### Desktop (Chrome/Edge)

#### 步骤:
1. 打开 GTD Focus 网站
2. 地址栏右侧出现 **安装** 图标 (⊕)
3. 点击安装图标
4. 点击 **"安装"**
5. 应用出现在桌面/开始菜单

#### 特点:
- 独立窗口运行
- 无浏览器地址栏
- 支持离线使用
- 可在任务栏固定

## 📡 离线功能

### 可用的功能
- ✅ 查看所有已加载的任务
- ✅ 添加新任务
- ✅ 编辑任务
- ✅ 删除任务
- ✅ 完成任务
- ✅ 切换分类
- ✅ 搜索过滤
- ✅ 切换主题

### 需要网络的功能
- ⚠️ WebDAV 同步 (上传/下载)
- ⚠️ 首次加载应用

### 离线工作原理
1. **首次访问** - 缓存核心文件 (HTML/CSS/JS)
2. **离线使用** - 从缓存加载，数据存本地
3. **恢复网络** - 自动更新缓存
4. **手动同步** - 点击 Sync 同步到云端

## 💾 数据存储

### 存储位置
- **任务数据:** localStorage (浏览器本地存储)
- **配置信息:** localStorage
- **缓存文件:** Cache Storage (Service Worker)

### 存储限制
| 浏览器 | 存储限制 |
|--------|----------|
| Chrome | ~6GB |
| Safari | ~1GB |
| Firefox | ~2GB |
| Edge | ~6GB |

### 数据持久化
- ✅ 关闭浏览器 - 数据保留
- ✅ 重启设备 - 数据保留
- ✅ 更新应用 - 数据保留
- ⚠️ 清除缓存 - 数据丢失
- ⚠️ 卸载应用 - 数据可能丢失

**建议:** 定期使用 WebDAV 同步备份数据

## 🔄 更新机制

### 自动更新
1. **检测更新** - Service Worker 检查新版本
2. **后台下载** - 静默下载新文件
3. **等待激活** - 关闭所有标签页后激活
4. **下次打开** - 使用新版本

### 手动更新
1. 刷新页面 (Ctrl+F5 / Cmd+Shift+R)
2. 强制刷新清除缓存

### 版本检查
打开浏览器控制台 (F12)，查看:
```
[PWA] SW registered: /
[PWA] New version available
```

## 🐛 常见问题

### Q1: 为什么安装后还是浏览器界面？
**A:** 某些浏览器/系统不支持 PWA，或安装方式不正确。确保:
- 使用 Chrome/Safari/Edge
- 按照正确步骤安装
- 系统版本较新

### Q2: 离线后无法使用？
**A:** 检查:
- 是否首次访问 (需先联网加载)
- Service Worker 是否注册成功
- 浏览器是否支持 PWA

### Q3: 数据会丢失吗？
**A:** 正常情况不会。但建议:
- 定期同步到 WebDAV
- 不要清除浏览器数据
- 导出重要数据备份

### Q4: 更新后数据没了？
**A:** PWA 更新不会影响 localStorage 数据。如果数据丢失:
- 检查是否清除了浏览器数据
- 从 WebDAV 恢复备份
- 检查浏览器隐私模式

### Q5: 桌面版和移动版数据同步？
**A:** PWA 本身不同步数据，需使用 WebDAV:
1. 设备 A 上传到 WebDAV
2. 设备 B 从 WebDAV 下载
3. 数据保持一致

## 🔧 开发者信息

### Service Worker
- **文件:** service-worker.js
- **策略:** 网络优先，缓存备用
- **缓存:** 核心文件 + 用户访问过的资源

### 缓存策略
```javascript
// 网络优先
fetch(request)
  .then(networkResponse => {
    // 更新缓存
    return networkResponse;
  })
  .catch(() => {
    // 返回缓存
    return cache.match(request);
  });
```

### 更新检测
```javascript
// 检查新版本
navigator.serviceWorker.ready.then(registration => {
  registration.update().then(() => {
    console.log('Checking for updates...');
  });
});
```
